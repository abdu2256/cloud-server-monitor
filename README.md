# Server Resource Monitoring & Logging System (Python)

This project is a Python-based server monitoring tool designed to simulate
basic cloud server health monitoring.

## Features
- Monitors CPU, RAM, and Disk usage
- Determines server health status based on thresholds
- Logs monitoring data with timestamps
- Continuous real-time monitoring behavior

## Technologies Used
- Python
- psutil library

## Use Case
This project demonstrates basic cloud and server monitoring concepts
commonly used in IT and Cloud Core environments.

## How to Run
python server_monitor.py
